# read-filepath-from-clipboard

Read file paths from clipboard, working on cross-platform

![demo](./demo.gif)

## API

### sync()

Returns file path array.

If file path not exist in clipboard, return empty array.

### async()

Returns file path array.

If file path not exist in clipboard, return empty array.
